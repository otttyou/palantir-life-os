# Life OS Workspace

A Notion-style editable Life OS for entrepreneurship planning, study systems, and generated operating outputs.

## Features
- Editable page-style workspace
- Structured founder brief and venture thesis inputs
- Generated operating brief, risk note, 30-day plan, and study output
- Long-term plan table and timeline
- JSON import and export
- Light and dark mode

## References
- Notion dashboards: https://www.notion.com/help/dashboards
- Notion layouts: https://www.notion.com/help/layouts
- Notion writing and editing basics: https://www.notion.com/help/writing-and-editing-basics

## Run locally
Serve the folder with any static file server and open index.html.
